四、字符串常见方法

a. 获取下标为 * 的字符 ( charAt( * ) )
b. 获取下标为 * 的字符的编码(ASCII码)（ charcodeAt(*) ）
c. ASCII编码转换成字符( fromCharCode() )
d. 连接字符串( concat() )
e. 字符串的查找方法（ .indexof() ）
f. 正则匹配 (返回第一次出现的位置) ( .search() )
g. 替换字符串( .replace() )
h. 截取字符串，范围是 [ start , end ) ( .substring( start, end ) )

！substring（） 填写字符串的首尾坐标，而 substr（）填写初始坐标，以及位数。
